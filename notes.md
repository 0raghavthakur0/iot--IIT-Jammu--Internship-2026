# Development Notes

## Main Branch
Contains the final stable version of the IoT Summer School assignment.

## Dev Branch
Used for developing and testing new features before merging into the main branch.

## Sensor Module Branch
Contains Arduino programs related to sensors such as DHT11, LDR, HC-SR04, PIR, and other sensor experiments.

## Wireless Module Branch
Contains projects related to Wi-Fi, Bluetooth, ESP8266, ESP32, MQTT, and other wireless communication modules.
