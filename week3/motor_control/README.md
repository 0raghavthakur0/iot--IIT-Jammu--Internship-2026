# DC Motor Speed Control using L298N

## Components

- Arduino UNO
- L298N Motor Driver
- DC Motor
- 10kΩ Potentiometer
- 2 Push Buttons
- Breadboard
- Jumper Wires

## Connections

ENA → Pin 9 (PWM)

IN1 → Pin 7

IN2 → Pin 8

Potentiometer → A0

Direction Button → Pin 2

Start/Stop Button → Pin 3

## Features

- Potentiometer controls motor speed (0–255 PWM).
- Direction button switches between Forward and Reverse.
- Start/Stop button enables or disables the motor.
- Serial Monitor displays motor direction, speed percentage, and current state.
