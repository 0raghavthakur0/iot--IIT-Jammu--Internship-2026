# Keypad + LCD Password Access

## Components

- Arduino UNO
- 4x4 Matrix Keypad
- 16x2 LCD
- Green LED
- Red LED
- Passive Buzzer
- Breadboard

## Default PIN

1234

## Features

- LCD displays "ENTER PIN:"
- Correct PIN → ACCESS GRANTED + Green LED
- Wrong PIN → ACCESS DENIED + Red LED + Buzzer
- System locks for 10 seconds after 3 incorrect attempts.
