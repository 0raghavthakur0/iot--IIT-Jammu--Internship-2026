# Q29 – Sensor Calibration

## What is Sensor Calibration?

Sensor calibration is the process of comparing a sensor's readings with a known standard and adjusting the sensor or software so that the readings become as accurate as possible.

Calibration reduces measurement errors and improves the reliability of sensor data.

---

## Why is Sensor Calibration Important in IoT?

Sensor calibration is important because:

- It improves measurement accuracy.
- It reduces sensor errors and drift.
- It increases the reliability of IoT systems.
- It provides consistent readings over time.
- It helps make better decisions based on sensor data.

For example, if a soil moisture sensor is not calibrated, it may report incorrect moisture values, causing plants to be overwatered or underwatered.

---

## Calibrating an Analog Soil Moisture Sensor

An analog soil moisture sensor gives a raw analog value between **0 and 1023**.

To convert this value into an accurate moisture percentage:

### Step 1 – Dry Calibration

Place the sensor completely in dry air.

Record the analog value.

Example:

```text
Dry Value = 850
```

---

### Step 2 – Wet Calibration

Place the sensor in water or fully wet soil.

Record the analog value.

Example:

```text
Wet Value = 350
```

---

### Step 3 – Convert to Percentage

Use the Arduino `map()` function.

Example:

```cpp
int moisture = map(sensorValue, 850, 350, 0, 100);
```

This converts the raw sensor value into a moisture percentage from **0% to 100%**.

---

# Two-Point Calibration Method

The two-point calibration method uses two known reference points:

- Dry condition (0% moisture)
- Wet condition (100% moisture)

These two values are used to calculate all intermediate moisture levels accurately.

This method is simple, reliable, and commonly used with analog sensors.

# Advantages

Sensor calibration is an essential process in IoT systems because it ensures that sensor readings are accurate and reliable.
