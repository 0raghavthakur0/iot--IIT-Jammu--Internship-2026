# Q12 – Microcontroller vs Microprocessor

| Parameter | Microcontroller (Arduino UNO) | Microprocessor (Raspberry Pi) |
|-----------|-------------------------------|-------------------------------|
| CPU Speed | 16 MHz | 1.5 GHz (Approx.) |
| On-chip RAM | 2 KB SRAM | Up to 8 GB RAM (External) |
| On-chip Flash | 32 KB | Uses microSD Card for storage |
| Operating System | No Operating System | Linux (Raspberry Pi OS) |
| Typical Use Case | Embedded systems, sensor control, automation | Computer vision, web server, AI, multimedia |
| Power Consumption | Very Low | Higher than microcontroller |

---

## Microcontroller

A microcontroller is a compact integrated circuit that contains a CPU, memory, and input/output peripherals on a single chip. It is mainly used in embedded systems where dedicated control of devices is required.

**Example:** Arduino UNO

### Advantages
- Low power consumption
- Low cost
- Easy to program
- Suitable for real-time applications

---

## Microprocessor

A microprocessor is a CPU that requires external RAM, storage, and peripherals to function. It is capable of running an operating system and performing complex computing tasks.

**Example:** Raspberry Pi

### Advantages
- High processing speed
- Supports multitasking
- Can run Linux
- Suitable for IoT gateways, AI, and image processing

---

## Conclusion

A microcontroller is ideal for controlling sensors, LEDs, motors, and embedded devices, while a microprocessor is suitable for applications requiring high computing power such as machine learning, web servers, and multimedia processing.
