# Digital Piano using Arduino

## Components

- Arduino UNO
- Passive Buzzer
- 5 Push Buttons
- Breadboard
- Jumper Wires

## Pin Connections

Button 1 → Pin 2

Button 2 → Pin 3

Button 3 → Pin 4

Button 4 → Pin 5

Mode Button → Pin 6

Buzzer → Pin 8

## Features

- Four musical notes
- Major/Minor mode
- Two-button chord (Sol)
- Serial Monitor displays current mode
